   - `/callout`

   - `/divider` para separar seções

3. **Usar Toggles:**
   - `/toggle` para esconder/mostrar detalhes
   - Exemplo: Esconder tabela de XP quando não precisar

4. **Colunas:**
   - Arrastar blocos lado a lado
   - Criar layout 2 ou 3 colunas
